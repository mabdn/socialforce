TrajNet++#

TrajNet++ is a pedestrian forecasting challenge [KKA21]. This notebook walks through a first attempt to fit to real-world data from this challenge [LCL07].

def trajnet_to_socialforce_scenario(pxy):
    pxy = torch.from_numpy(pxy)
    velocities = (pxy[1:] - pxy[:-1]) * 2.5  # convert to m/s with FPS
    states = torch.full((pxy.shape[0], pxy.shape[1], 4), float('nan'))
    states[:, :, :2] = pxy
    states[:-1, :, 2:4] = velocities
    states[-1, :, 2:4] = velocities[-1]

    return torch.stack([simulator.normalize_state(state) for state in states], dim=0)

MLP#

We infer the parameters of an MLP to approximate the 1D scalar function \(\textrm{SF}(b)\) above from synthetic observations. The PedPedPotentialMLP is a two-layer MLP with softplus activations:

(14)#\[\begin{align} \textrm{MLP}(b) &= \textrm{Softplus} \;\; L_{1\times5} \;\; \textrm{Softplus} \;\; L_{5\times1} \;\; b \end{align}\]

which is written in terms of linear and non-linear operators where the Softplus operator applies the softplus function on its input from the right and \(L\) is a linear operator (a matrix) with the subscript indicating the \(\textrm{output features} \times \textrm{input features}\). This two-layer MLP with 5 hidden units has 10 parameters.
